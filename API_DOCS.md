# 🔌 VyapaarVaani API Documentation

All API endpoints are prefixed with `/api`.
Base URL (Local): `http://localhost:8000`

---

## 🔐 Authentication & Headers

Every request (except public health checks) requires:
1. **Authorization**: `Bearer <Firebase_ID_Token>`
2. **X-Shop-Id**: The Firestore Document ID of the active shop.

---

## 🎙️ Ingestion API

### `POST /api/ingest`
Processes voice narration into structured ledger entries.

- **Request Type**: `multipart/form-data`
- **Body**: 
  - `file`: Audio file (WebM/WAV/MP3).
- **Process**:
  1. Transcribes Hinglish audio via Deepgram Nova-3.
  2. Extracts entities (Items, Expenses, Earnings) via Llama 3.3.
  3. Validates data against the vendor's business profile.
  4. Persists record to Firestore and invalidates health cache.

**Response (200 OK):**
```json
{
  "id": "ledger_doc_id",
  "transcript": "Aaj 20 kilo aam becha 40 ke bhav se...",
  "ledger_entry": {
    "items_sold": [{"name": "Aam", "qty": 20, "price": 40, "is_anomalous": false}],
    "expenses": [],
    "earnings": 800,
    "flags": []
  }
}
```

## 📊 Business Intelligence

### `GET /api/insights`
Calculates local ML insights (Revenue trends, Top items, Stock suggestions).

- **Features**: 6-hour Firestore cache.
- **Logic**: Uses Linear Regression for forecasting and Fuzzy Matching for normalization.

**Response (200 OK):**
```json
{
  "status": "ok",
  "revenue_trend": { ... },
  "top_items": [ ... ],
  "stock_suggestions": [ ... ],
  "confidence_score": 85,
  "days_recorded": 12
}
```

## 🩺 Financial Health

### `GET /api/health-score`
Computes a 0-100 score based on 5 creditworthiness criteria.

- **Criteria**: History, Earnings Consistency, Expense Stability, Diversification, Cleanliness.
- **Features**: 12-hour Firestore cache.

**Response (200 OK):**
```json
{
  "score": 78,
  "tier": "Good",
  "loan_estimate": 45000,
  "criteria": [ ... ],
  "gaps": ["Record 3 more days to reach next tier"]
}
```

## 📝 Ledger History

### `GET /api/ledger`
Retrieves the last 7 days of business activity.

### `DELETE /api/ledger/{entry_id}`
Deletes a specific record. Verified via UID check.

### `PATCH /api/ledger/{entry_id}`
Resolves ambiguous data flags generated by the AI engine.

---

## 🛍️ Shop Management

### `GET /api/shops`
Lists all shops owned by the user.

### `POST /api/shops`
Creates a new business profile (Name, Type, City).

### `PATCH /api/shops/{shop_id}`
Updates shop settings.

---

## 🛠️ Utilities

### `POST /api/translate`
Utility to translate complex Hinglish narration into professional business English for reporting.
